# Game-Off 2017

Team Kickpunch's rocking entry ...

## Development

### Requirements

* [Node 8.6.0](https://nodejs.org)
* [npm 5.5.1](https://www.npmjs.com/package/npm)

### Installation

```
npm i
```

### Running a dev server

```
npm start
```
Default address is <http://0.0.0.0:3000>

### Get a production build
```
npm run build
```
